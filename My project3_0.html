Who am I

M y name is Julia Zubareva.

I am a junior Frontend Developer. A few years ago I used to work as junior web analyst, but the more I was doing web-analytics, the more curious I was about the process of web-product creation itself. Eventually I realised, that it would be interesting for me to start developing web projects and see how it works from the inside. So I made this decision and currently you're on the webpage of my first front-end project. Welcome!

What I do

Frontend Development

As a junior specialist I'm focused on learning essential set of front-end technologies: HTML5 and CSS3. In addition to that I'm using Boostrap framework and making first steps in JavaScript with help of jQuery. When I was developing this website I aimed to make it compliant with the latest trends in web development, such as responsive design, long scrolling, hamburger menu, flat minimalism. See below

Google Analytics

Comprehensive understanding of online product audience is one of essential keys for success. Google Analytics is a de-facto standard for nowadays Web analytics - and there are a lot of good reasons for its popularity. Although I'm aiming to change my career focus to front-end development, I believe that GA background provides a good basis for Web product development. See for details on my Google analytics experience below.

Development process

I treated my portfolio as a product. While creating it I wasn't just developer but also product manager: I defined goals, development plan and deadlines. To keep the process structured I used a couple of tools such as bugtracker (this time I went for Google docs) and version control system - Git.
